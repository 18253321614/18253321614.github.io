光伏发电原理

发表于 2026-01-15 分类于 新能源 本文字数： 882 阅读时长 ≈ 1 分钟

光伏发电是利用的半导体界面的光生伏特效应，有光直接产生电能。

1 硅晶体

首先是半导体，一般采用硅，因为地球储量丰富。但是需要提取单晶硅。硅的结构特点是原子外围有3层工14个电子，只有第三层也就是最外围的4个电子参与导电，在单晶硅中，每个硅原子和周围的四个硅原子通过电子结合，两个电子形成一个共价键，非常稳定，所以在绝对零度下，硅晶体是绝缘的，但是在常温下，会形成一些离散的电子，具有一定的导电性，且温度越高，导电性越强。

传统的LSTM、XGBoost或Transformer变体（如Informer）对数值型数据（历史功率、辐照度、温度）处理极佳，但对文本型数据（运维日志、天气预警文本、设备检修单）无能为力。例如，如果运维人员提交了一条“3号汇流箱下午2点检修”的文本记录，传统模型无法读取，导致预测偏高。LLM可以理解非结构化数据（如“受台风外围气流影响，午后可能有短时强降水”），同时结合RAG知识增强，检索该电站的历史类似气象日的表现、检索电站的物理拓扑信息、检索最新的设备维护手册或日志，完成数据修正、结果解释、可视化展示、运维辅助、智能交互等功能。

地质时代简单梳理

发表于 2023-01-30 分类于 求知 本文字数： 475 阅读时长 ≈ 1 分钟

地质时代是指地球上各种地质事件发生的时代。

  1. 从大到小分为宙、代、纪、世、期。比如我们常说的侏罗纪，就是属于显生宙中生代侏罗纪；寒武纪属于显生宙古生代。
  2. 冥古宙、太古宙、元古宙、显生宙，从距今5.41亿年开始就是显生宙了（现在也是），显生宙的标志就是古生代寒武纪生物大爆发。其中前三个时期又称为隐生宙，因为当时生命还很原始。
  3. 将显生宙细看开来，分为古生代-中生代-新生代
    • 古生代：距今2.25亿年-约6亿年前，脊椎海洋生物开始繁盛，藻类、鱼类繁盛，两栖类登录，蕨类植物登录。
      • 寒武纪：5-6亿年前
      • 奥陶纪：4.4-5亿年前
      • 志留纪
      • 泥盆纪
      • 石炭纪
      • 二叠纪
    • 中生代：0.7亿-2.25亿年前，爬行动物的时代，裸子植物
      • 三叠纪
      • 侏罗纪：1.35-1.8亿年前
      • 白垩纪：0.7-1.35亿年前，恐龙灭绝，白垩纪结束，地质进入新生代。
    • 新生代：从7000万年前开始，哺乳动物的时代，被子植物繁盛。
      • 第三纪
      • 第四纪：它从约260万年前开始，一直延续。
        1. 更新世
        2. 全新世：开始于1.2-1万年前，直到现在。
  4. 每个纪的生物都不太一样，而化石是识别地层的头号指标，因此纪是最常用的地址单位。

简单了解后期在补充。
